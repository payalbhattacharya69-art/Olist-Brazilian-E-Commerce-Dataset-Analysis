# 📊 Olist Brazilian E-Commerce Data Analysis  

## 📌 Project Overview

This project analyzes customer behavior, sales trends, payment preferences, delivery performance, and customer satisfaction using the **Olist Brazilian E-Commerce Dataset**.

The goal is to extract meaningful business insights and identify opportunities to improve customer experience and overall business growth using **Python** and **Power BI**.

## 📁 Dataset Information

Olist Brazilian E-Commerce Dataset  

### 📦 Tables Used:
- Customers Dataset  
- Orders Dataset  
- Order Items Dataset  
- Payments Dataset  
- Reviews Dataset  
- Products Dataset  

## 🛠 Tools & Technologies

- Python 🐍  
- Pandas  
- Matplotlib  
- Seaborn  
- Power BI 📊  

## 🧹 Data Preparation

- Imported multiple CSV files using Pandas  
- Handled missing values and data types  
- Converted date columns into datetime format  
- Merged datasets using `customer_id` and `order_id`  
- Created a unified dataset for analysis  

## 📊 Exploratory Data Analysis (EDA)

### 📈 Order Trend Analysis
- Orders showed a steady increase over time  
- Indicates growing customer engagement  

### 💳 Payment Method Analysis
- Credit cards were the most preferred payment method  
- Shows preference for flexible and installment payments  

### ⭐ Review Score Analysis
- Most customers gave high ratings  
- Indicates overall positive customer satisfaction  

### 🛍 Top Product Categories
- Few categories contributed major share of sales  
- Highlights need for focused marketing strategy  

### 🔁 Repeat vs One-Time Customers
- Majority were one-time buyers  
- Indicates low customer retention rate  

## 🧪 Hypothesis Testing

### 1️⃣ Faster Deliveries Improve Reviews
- Faster deliveries correlate with higher review scores  
- ✅ Hypothesis supported  

### 2️⃣ Credit Card Users Spend More
- Credit card users show higher average transaction value  
- ✅ Hypothesis supported  

## 🔑 Key Insights

- Orders increased consistently over time  
- Credit cards dominated payment methods  
- Delivery speed impacts customer satisfaction  
- Low customer retention rate  
- Few product categories drive most revenue  
- Overall customer satisfaction is positive  

## 💡 Business Recommendations

- 🎯 Improve customer retention with loyalty programs  
- 🚚 Optimize delivery operations for faster shipping  
- 📦 Focus on high-performing product categories  
- 💳 Promote credit card offers and EMI options  
- 📞 Improve customer support and complaint handling
